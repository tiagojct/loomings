Loomings

The shapes loom before they take form.

A minimalist Markdown writing app for macOS. Real syntax highlighting, focus mode, distraction-free.

macOS · Windows · Linux · unsigned builds

macOS: "Loomings.app is damaged"?

Unsigned builds get quarantined by Gatekeeper. After moving the app to /Applications:

xattr -dr com.apple.quarantine /Applications/Loomings.app

Or open System Settings → Privacy & Security → "Open Anyway" after a failed launch attempt.

What it does

Inline Markdown

Bold, italic, headings, code, links — rendered as you type. CodeMirror 6 with lezer markdown.

Focus mode

Dims everything outside the current sentence. Cmd+Shift+D.

Outline palette

Cmd+P — jump to any heading by fuzzy match.

Smart typography

"foo" → "foo", -- → —, ... → … as you type. Toggleable.

Lists that finish themselves

Enter on - foo continues the bullet. Numbered lists increment. Empty line exits.

Word goal

Set a target. Progress shows in the statusbar. Cmd+Shift+G.

File watcher

If a file changes outside the app, prompts to reload. No silent overwrites.

Crash recovery

Unsaved buffer persisted to disk every second. Recover on next launch.

Pequod theme

Dark navy + amber, parchment for light mode. Follows system or toggle manually.

Under the hood

  • Tauri v2 — Rust shell + system WebKit. No bundled Chromium.
  • CodeMirror 6 — virtualized editor, native undo, IME-safe.
  • markdown-it — CommonMark preview with URL sanitization.
  • ~12 MB universal binary. ~30 MB idle RAM.
  • MIT licensed. Open source.
